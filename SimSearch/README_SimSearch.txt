Instruction on Running by Command Line to Find the most kth similar TS of a given TS
-------------------------------------------------------------------------------------
Usage:
    ./run.sh PARAM1 PARAM2
    PARAM1: file name that to be compared, in top level folder
    PARAM2: top PARAM2 similar timeseries

Example:
    ./run.sh ts_data_1.txt 10
